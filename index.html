货品供给策略框架

小红书商业化运营 · 大健康赛道适用
Framework Purpose

用数据找到「该引什么货」：
需求 × 供给 双维度赛道诊断

不同赛道的用户决策路径不同——有人主动搜索，有人刷到就买。 我们用搜索量 + 阅读量 + 站外GMV衡量真实需求强度， 用动销SPU数 + 价格带覆盖 + 供给集中度衡量平台供给满足程度， 两轴交叉定位每个子赛道，输出「该引 / 持续 / 观察 / 清退」四类策略动作。

核心假设验证：在高需求 × 低供给赛道（A象限），SPU 增长与 GMV YoY 增长的相关性最强—— 这是招商资源优先投入的依据。
分析两步走
01

供需水位诊断

四维气泡图：这个赛道在哪里？
  • X轴（需求强度）：搜索量×40% + 阅读量×40% + 站外GMV×20%
  • Y轴（供给满足度）：动销SPU数×40% + 价格带覆盖×35% + (1-CR5)×25%
  • 气泡大小：站内 GMV 体量（资源分配优先级）
  • 气泡颜色：站外品牌渗透率——红色 = 大量品牌还在外面，是招商机会
02

供给策略决策

四象限输出四类策略动作
  • A（高需求·低供给）：主动招商，优先引入站外头部品牌，颜色越红越紧迫
  • B（高需求·高供给）：维持供给，精细化扶持头部 SPU，防止内卷
  • C（低需求·高供给）：内容种草破圈激活需求，暂不扩供给
  • D（低需求·低供给）：停止扩供，清退低效 SKU 或引导差异化
供给策略四象限
↑ 供给满足度
C · 供给过剩

🔴 内卷 · 红海

货已经进来很多，但需求未被激活。SPU 增速跑不赢 GMV 增速，价格战拉低整体效率。小红书特有现象：好货缺内容承接。
→ 内容种草破圈激活需求，暂不扩供给
示例：基础保健品（维矿白牌）、同质化器械
B · 持续做

🔵 均衡 · 成熟

需求旺盛，供给充足且结构健康。SPU → GMV 相关性已验证。核心任务是提升品质，扶持头部，防止进一步内卷。
→ 维持供给，精细化扶持头部 SPU
示例：隐形眼镜、普通矿维综合营养
D · 观察

⚪ 冷启动 · 待验证

需求和供给都不足，市场尚未成熟。可能是早期教育期品类，也可能是伪需求。SPU 增长不带动 GMV，暂不投入资源。
→ 停止扩供，清退低效 SKU 或引导差异化
示例：nmn、谷胱甘肽、睡眠管理营养品
A · 该引没引

🟢 供给不足 · 蓝海

需求强劲但供给严重不足。站外有大量品牌未入驻（颜色越红招商机会越大），每 SPU 承接的需求密度极高。
→ 主动招商，优先引入站外头部品牌
示例：滋养食补、养生理疗、健康手环/手表

📊 关键指标说明

需求侧（X轴成分）
搜索量（绝对值）
赛道关键词搜索 UV，来源：灵犀搜索洞察。主动意图信号，权重最高
高 → 需求规模大
笔记阅读量（绝对值）
赛道笔记总阅读 PV，来源：灵犀内容洞察。捕捉「刷到即买」的短决策链路需求（食饮类权重需上调）
高 → 被动种草需求旺
站外 GMV 规模
淘系/抖音该赛道 GMV，来源：魔镜/生意参谋。代表行业真实需求盘子大小
高 → 行业盘子大
供给侧（Y轴成分）
动销 SPU 数
有交易的商品数量（绝对值归一化），来源：RedBI。基础供给门槛
低 → 供给量不足
价格带覆盖度
5个价格段（<50/50-200/200-500/500-1500/>1500元）中，有≥3个动销SPU的段数÷5
低 → 价格段有空白
1 − CR5（分散度）
1减去前5个SPU的GMV占比。越接近1越分散，说明没有单一爆款垄断，供给结构健康
低（CR5高）→ 供给过度集中
辅助诊断指标（气泡颜色 + 标注）
站外品牌渗透率
站内已入驻品牌数÷全网活跃品牌数。来源：人工统计或BD信息。<30% = 大量品牌在外面没进来
低 → 主动招商机会
GPM（赛道均值）
GMV÷曝光量×1000。来源：RedBI/聚光。供需匹配效率的结果校验，不放入轴成分（避免与需求/供给混同）
持续走低 → 匹配效率差
好评率 / 退货率
来源：商家后台或运营BD信息。反映供给质量健康度，辅助判断D象限供给是否需要清退
退货率高 → 供给质量差
