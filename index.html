货品供给策略框架

小红书商业化运营 · 大健康赛道适用
Framework Purpose

我们在回答一个问题：
应该引入什么货品，不应该引入什么？

小红书作为种草平台，天然积累了大量用户需求信号（搜索行为、内容互动）。 我们希望用这些数据，系统性地判断哪些子赛道供给不足、哪些已经内卷， 从而指导招商策略和供给调控动作。

核心假设： 在需求旺盛但供给不足的赛道，引入新的品牌/SKU（SPU增长）能够带来 GMV 的 YoY 正向增长。 这不一定是严格因果关系，但可以通过数据验证其相关性强弱。
分析三步走
01

行业趋势诊断

这个行业行不行？
  • 外部市场规模增速（第三方数据）
  • 别的平台行不行（抖音/淘宝 GMV 增速）
  • 小红书站内搜索量 YoY
  • 笔记互动量 YoY
02

供需水位诊断

我们的供给够不够？
  • 需求：搜索渗透率 / 购买用户渗透率
  • 供给：动销 SPU 数 / 品牌数
  • 匹配：搜索→购买转化率
  • 效率：GMV / SPU（每 SPU 产出密度）
  • 竞争：品牌数 / 搜索量占比
03

供给策略决策

该怎么调？
  • 该引没引 → 主动招商
  • 引了表现好 → 持续扶持
  • 引了表现差 → 改策略
  • 该清退 → 清退（违规/严重内卷）

🟢 供给不足 · 蓝海

行业高增长，搜索需求旺，但平台品牌数少、SKU 稀缺。每个 SKU 承接大量需求，GMV 密度极高。
→ 主动招商，引入新品牌 / 新 SKU
大健康示例：女性健康（高端）、睡眠健康、银发健康
B · 持续做

🔵 均衡 · 成熟

行业高增长，供给数量合理，转化率稳定，SPU → GMV 相关性验证成立。
→ 维持供给，提升品质，扶持头部
大健康示例：运动健康 / 智能健身设备
C · 待观察

⚪ 冷门 · 待激活

行业增速低，搜索需求弱，供给也少。可能是需求尚未被内容激活，或是真正的小众赛道。
→ 观察，不主动扩供给
需根据实际数据识别，定期复盘
D · 该改 / 该清

🔴 内卷 · 红海

供给数量大，但 GMV 增速跑不赢 SPU 增速。价格战加剧，GMV 密度持续下滑，CPE 上升。
→ 停止扩供，引导差异化 / 清退
大健康示例：基础保健品（维矿）、个人护理 / 医美
◀ 供给水位低 供给水位高 ▶
