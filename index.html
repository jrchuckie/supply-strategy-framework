货品供给策略框架

小红书商业化运营 · 大健康赛道适用 · v3
Framework Purpose

用数据找到「该引什么货」：
需求 × 供给 双维度赛道诊断

三步走——先用赛道打标把货品归到正确的赛道，再用搜索量 + 阅读量衡量真实需求强度， 用动销SPU数 + SPU增速衡量供给满足程度， 两轴交叉定位每个子赛道，输出「该引 / 持续 / 观察 / 清退」四类策略动作。

⚠️ 核心问题：搜索量高可能是某个单一SPU驱动的——这对判断「要不要引入这个SPU」有帮助，但对判断「这个赛道值不值得投」是噪音。 我们需要在 Step 0 识别并剥离单SPU驱动效应，还原赛道级别的真实需求。
Step 0 — 货品赛道打标（新增）

🏷️ 先把货品归对赛道，再做供需诊断

在分析需求与供给之前，先确定每个 SPU 属于哪个赛道——否则后续所有指标都可能被"错位归属"污染。
STEP 0.1

基于搜索词的赛道映射

用灵犀搜索洞察的关键词-赛道对应关系，为每个 SPU 打标。一个 SPU 可能命中多个赛道词，取搜索 UV 最高的赛道作为主归属。

→
STEP 0.2

SPU 集中度校验

检查每个赛道内 Top 1 SPU 的搜索量占比。如果 Top 1 SPU 占了赛道搜索量的 50%+，说明这个赛道的"需求信号"很可能是单爆款驱动的，需要降权处理。

→
STEP 0.3

噪音赛道标记

被识别为单 SPU 驱动的赛道，打上「⚠️ SPU驱动」标签。这类赛道的供需分析结论仅供参考 SPU 引入决策，不作为赛道级别的招商依据。

⚡ 为什么需要这一步？——"搜索量高 ≠ 赛道需求旺"

举例：某赛道搜索量 1000 万，但其中 800 万来自同一个 SPU（比如某个爆款益生菌）。这说明：

  • 对 SPU 引入决策有用：这个爆款确实有需求，可以考虑引入
  • 对赛道判断是干扰：除去这个 SPU，赛道真实需求只有 200 万，可能根本不值得投入招商资源
  • 处理方式：在计算赛道级搜索量时，用「Top 3 SPU 搜索量」或「去 Top 1 后搜索量」作为修正指标，避免单爆款扭曲赛道判断

供需水位诊断

四维气泡图：这个赛道在哪里？
  • X轴（需求量）：搜索量 + 搜索量YoY + 阅读量 + 阅读量YoY
  • Y轴（供给满足度）：动销SPU数 + SPU YoY
  • 气泡大小：站内 GMV 体量
  • 气泡颜色：站外品牌渗透率——红色 = 招商机会
02

供给策略决策

四象限输出四类策略动作
  • A（高需求·低供给）：主动招商，优先引入站外头部品牌
  • B（高需求·高供给）：维持供给，精细化扶持头部 SPU
  • C（低需求·高供给）：内容种草破圈，暂不扩供给
  • D（低需求·低供给）：停止扩供，清退低效 SKU
供给策略四象限
↑ 供给满足度
C · 供给过剩

🔴 内卷 · 红海

货已经进来很多，但需求未被激活。SPU 增速跑不赢 GMV 增速，价格战拉低整体效率。小红书特有现象：好货缺内容承接。
→ 内容种草破圈激活需求，暂不扩供给
示例：基础保健品（维矿白牌）、同质化器械
B · 持续做

🔵 均衡 · 成熟

需求旺盛，供给充足且结构健康。SPU → GMV 相关性已验证。核心任务是提升品质，扶持头部，防止进一步内卷。
→ 维持供给，精细化扶持头部 SPU
示例：隐形眼镜、普通矿维综合营养
D · 观察

⚪ 冷启动 · 待验证

需求和供给都不足，市场尚未成熟。可能是早期教育期品类，也可能是伪需求。SPU 增长不带动 GMV，暂不投入资源。
→ 停止扩供，清退低效 SKU 或引导差异化
示例：nmn、谷胱甘肽、睡眠管理营养品
A · 该引没引

🟢 供给不足 · 蓝海

需求强劲但供给严重不足。站外有大量品牌未入驻（颜色越红招商机会越大），每 SPU 承接的需求密度极高。
→ 主动招商，优先引入站外头部品牌
示例：滋养食补、养生理疗、健康手环/手表

📊 关键指标说明

需求侧（X轴成分）
搜索量（绝对值）
赛道关键词搜索 UV，来源：灵犀搜索洞察。主动意图信号，最核心需求指标
高 → 需求规模大
搜索量 YoY
同比增速，判断需求是在扩张还是收缩。搜索量高但 YoY 负增长 = 需求萎缩中
正 → 需求增长中
笔记阅读量（绝对值）
赛道笔记总阅读 PV，来源：灵犀内容洞察。捕捉「刷到即买」的短决策链路需求
高 → 被动种草需求旺
阅读量 YoY
阅读量的同比增速，判断内容侧的需求趋势。高搜索量 + 负阅读 YoY = 搜索意图强但内容供给不足
正 → 内容需求增长
供给侧（Y轴成分）
动销 SPU 数
有交易的商品数量（绝对值归一化），来源：RedBI。基础供给门槛
低 → 供给量不足
SPU YoY
动销 SPU 数的同比增速。判断供给是在扩张还是收缩。SPU 数多但负增长 = 供给在撤退
正 → 供给扩张中
Step 0 辅助指标（赛道打标 & 噪音识别）
Top 1 SPU 搜索占比
赛道内搜索量最大的 SPU 占赛道总搜索量的比例。>50% 标记为「⚠️ SPU驱动」
高 → 单爆款驱动噪音
Top 3 SPU 搜索占比
前三大 SPU 的搜索量占比。越高说明赛道需求越集中，头部效应越强
高 → 需求集中度偏高
去 Top 1 搜索量
赛道总搜索量减去 Top 1 SPU 搜索量。用于剥离单爆款效应，还原赛道真实需求
低 → 真实需求可能偏弱
辅助诊断指标（气泡颜色）
站外品牌渗透率
站内已入驻品牌数÷全网活跃品牌数。<30% = 大量品牌在外面没进来
低 → 主动招商机会
